# FUTURE_FS_01 | Full-Stack Developer Portfolio

## 🚀 Overview
A comprehensive full-stack portfolio application designed to showcase modern web development practices. This project serves as a central hub for my professional identity, featuring a dynamic frontend and a scalable backend architecture.

## 🛠️ Tech Stack
### Frontend (client)
- **Framework:** React.js (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Path Mapping:** JSConfig Alias (`@/*`)

### Backend (server)
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** Supabase (PostgreSQL)
- **ORM:** Prisma

## 📂 Project Structure
This repository is organized as a **Monorepo**:
- **/portfolio-frontend**: The React frontend, optimized for performance and responsive design.
- **/portfolio-backend **: The Node.js API handling data persistence and server-side logic.
